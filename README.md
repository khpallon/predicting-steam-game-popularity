# predicting-steam-game-popularity

Dataset 1 (37.89 MB): This dataset includes data on the average and peak monthly player counts for 6725 unique games available on Steam (2020-2025) [1]  
Dataset 2 (1.07 GB): This dataset contains additional data on the games in dataset 1 such as the publisher, price, etc. [2]  
Goal 1: Collect and clean the data so it can be used for statistical analysis.  
Goal 2: Find out which factors have the biggest impact on how popular a game becomes over time.  
Goal 3: Build a model that can use past player data to predict future trends.  
The main objective is to use data analysis and predictive modeling to identify patterns and forecast how game popularity changes on Steam.  
Links:  
[1] [steam monthly average players](https://www.kaggle.com/datasets/lunthu/steam-monthly-average-players)  
[2] [steam all games data](https://www.kaggle.com/datasets/fmpugliese/steam-all-games-data?select=steam_app_data.csv)
